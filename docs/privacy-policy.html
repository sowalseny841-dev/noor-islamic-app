🌙 Noor نور — Politique de Confidentialité

Dernière mise à jour : 20 avril 2026

1. Données collectées

Noor collecte uniquement les données nécessaires au fonctionnement de l'application :

  • Localisation GPS : utilisée uniquement pour calculer les horaires de prière et la direction de la Qibla. Non stockée sur nos serveurs.
  • Préférences locales : paramètres de l'application (méthode de calcul, thème, langue) stockés localement sur votre appareil.

2. Données NON collectées

Noor ne collecte jamais :

  • Données personnelles (nom, email, téléphone)
  • Données de paiement
  • Historique de navigation
  • Contacts ou fichiers
  • Données biométriques

3. Permissions requises

  • ACCESS_FINE_LOCATION : calcul des prières et Qibla
  • INTERNET : téléchargement des données Coran et audio
  • RECEIVE_BOOT_COMPLETED : rétablissement des notifications après redémarrage
  • VIBRATE : retour haptique pour le tasbih

4. Partage des données

Aucune donnée n'est partagée avec des tiers. L'application ne contient aucune publicité et aucun SDK de tracking.

5. Sécurité

Toutes les données locales sont stockées de manière sécurisée sur votre appareil via les mécanismes standards d'Android.

6. Enfants

Noor est adapté à tous les âges. Aucune donnée n'est collectée sur les utilisateurs mineurs.
